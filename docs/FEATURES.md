# Features

## Implemented

- FastAPI backend
- React and Vite frontend
- MongoDB user persistence
- JWT signup and login
- bcrypt password hashing
- Host, participant, and admin roles
- Profile metadata with language, pronouns, and voice preference
- Room join and leave workflow
- WebSocket room chat
- Session IDs for connected clients
- Per-client outbound delivery queues
- Direct messages
- Host broadcast messages
- Participant list
- Room statistics
- LibreTranslate integration
- Translation caching
- Same-language translation skipping
- Structured translation logging
- Faster-Whisper speech-to-text
- Piper TTS synthesis
- Voice routing service
- Voice verification page
- WebRTC audio signaling
- WebRTC video foundation
- Local and remote video rendering
- Mic and camera toggles
- WebRTC diagnostics panel
- HTTPS local development scripts
- Cross-device testing guide

## In Progress

- Real-time translated audio playback workflow
- Better sentence segmentation for STT
- More reliable cross-device media recovery
- Per-language Piper voice expansion
- Translation latency optimization
- Admin dashboard UI

## Planned

- Automatic translated audio listener modes

- Meeting recording and transcript export
- Room persistence in MongoDB

- Role-based meeting moderation controls
